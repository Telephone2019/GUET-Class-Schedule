# 将GUET课表转为可导入的日历文件

已经厌倦了超级课程表？

想要尝试将课程表导入日历中？

想要体验日历的上课准时提醒？

来试试这个小工具吧，它可以将GUET教学管理系统网站上的课表转为ics文件以供导入日历。

## Tip

- 目前仅支持**Windows**平台。
- 软件已经可以使用，但还有许多功能需要完善。
- 目前可以通过重新编译来更改一些自定义参数，<u>重新编译时源字符集使用**UTF-8**，执行字符集务必使用**UTF-8**</u>。
- 不想重新编译的小伙伴也可以直接使用根目录下的`GUET-Class-Schedule.exe`。
- 如果想要将程序拷贝到别处，请将同一目录下的`GnuWin32`文件夹一同拷贝，否则程序将无法正常运行。
- 需要使用方法的小伙伴可以咨询作者`telephone.tan@qq.com`。
- 有这方面需求的小伙伴可以通过`创建issue`/`给作者发送电子邮件`吸引作者的注意，作者会视情况完善/增加更多的功能。
